Origem do nome Java

O nome do famoso linguagem de programação Java tem uma origem curiosa. Os criadores do Java eram grandes fãs de café, e "Java" é um dos termos usados para se referir ao café em algumas partes do mundo. O logo da linguagem, inclusive, é uma xícara de café fumegante.

Tabela Linguagens de Programação e seus Lançamentos:

ID Nome Linguagem data de Lançamento Criador Descrição
1 C++ 1985 Bjarne Stroustrup C++ é uma extensão da linguagem C, adicionando recursos de programação orientada a objetos. É amplamente utilizada em desenvolvimento de sistemas, jogos, e software de alto desempenho.
2 Java 1995 James Gosling (Sun Microsystems) Java é uma das linguagens mais populares e portáveis, sendo usada para desenvolvimento de aplicativos móveis (Android), sistemas corporativos, e aplicações web.
3 Python 1991 Guido van Rossum Python é uma linguagem de programação de alto nível, conhecida por sua sintaxe simples e legibilidade. É amplamente usada em ciência de dados, inteligência artificial, automação, desenvolvimento web e mais.
4 JavaScript 1995 Brendan Eich (Netscape) JavaScript é a principal linguagem para desenvolvimento web front-end, usada para criar páginas interativas. Com o Node.js, também é utilizada no back-end.
5 TypeScript 2012 Microsoft TypeScript é um superconjunto de JavaScript, adicionando tipagem estática. Ela é frequentemente usada em grandes projetos JavaScript, especialmente para aplicações web.
